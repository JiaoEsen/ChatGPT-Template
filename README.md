# ChatGPT-Template

**ChatGPT-Template** 是一款由**Vue3+TypeScript+Vite**等前端前沿技术打造的**静态模板**。

支持 **Markdown** 语法展示

## 启动方式
```js
   git clone https://github.com/JiaoEsen/ChatGPT-Template.git
   cd chatgpt-template
   pnpm install
   pnpm run dev
```
